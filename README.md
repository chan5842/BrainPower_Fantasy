# BrainPower_Fantasy
2019년 게임프로그래밍 강의 팀프로젝트

# 제작 기간
- 2019. 04. 30 ~ 2019. 06. 04(약 1개월)

# 조작키
- Enter : 대사 넘기기, 답 제출
- ESC : 게임 종료(버그로 인해 종료 되지 않는 씬이 존재함)
- SpaceBar : 디버그용 스킵 키
- 숫자 키 : 정답 입력 키
